# GrimfallTalentOverview
![GrimfallTalentOverview Preview](https://i.imgur.com/vl4h44n.png)

A lightweight Grimfall-WoW addon that consolidates all your active talents into a single view.

## How to Use

1. Open your Talents (default key: **N**).
2. The overview will appear on the right side of the UI.

## Installation


1. Download the Latest here [DOWNLOAD]https://github.com/Snickers213/GrimFall-Talentlist/releases/download/Release/GrimfallTalentOverview.7z
2. Place the `GrimfallTalentOverview` folder into your `\Grimfall\Games\Interface\AddOns` directory.
